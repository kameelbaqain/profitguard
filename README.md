# The Profit Guard

A private, internal B2B Progressive Web App for tracking supplier prices, detecting cost inflation, and protecting profit margins.

## Features

- Mobile-first dark UI with iOS safe-area spacing
- English and Arabic localization with RTL layout support
- Local IndexedDB persistence for suppliers, items, and invoices
- Jordanian Dinar formatting in English and Arabic
- Invoice entry workflow with a 5% price-spike intercept modal
- Dashboard metrics, recent invoices, supplier item lists, and price trends
- PWA manifest and app icons

## Run Locally

```bash
npm install
npm run dev
```

The app will be served at `http://127.0.0.1:5173/`.

## Build

```bash
npm run build
```
